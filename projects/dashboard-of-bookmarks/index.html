Bookmark Dashboard

A clean, modern Chrome extension that replaces your new tab with an organized, searchable bookmark dashboard. Browse by folder, search by title or URL, switch between grid and list view.

View on GitHub

Features

Browse by Folder

All Chrome bookmarks organized by folder hierarchy.

Search

Search bookmarks by title or URL.

Grid & List View

Switch between grid and list display modes.

Dark / Light Theme

Choose your preferred theme.

Clock & Greeting

Optional clock with personalized greeting.

Custom Display Name

Set your name for the greeting.

Dashboard

Clean, organized view of your bookmarks. Browse by folder or search across all.

Settings

Display Name, Theme (dark/light), Display Mode (grid/list), Clock toggle.

Install (Developer Mode)

  1. Clone the repository
  2. Open chrome://extensions/
  3. Enable Developer mode (top-right toggle)
  4. Click Load unpacked and select the project folder
  5. Open a new tab
