Nexus Tab

Your daily command center. A Chrome extension that replaces your new tab with a modular productivity dashboard—glassmorphism cards, customizable widgets, and preset layouts.

Install from Chrome Web Store

Widgets

Clock & Greeting

Live time, date, and personalized greeting.

Search Bar

Search the web or enter URLs directly.

Tasks

Full task management with kanban board and list views.

Quick Links

Customizable link grid with favicons.

Bookmarks

Browse and search Chrome bookmarks with tree navigation.

Notes

Persistent scratchpad for quick notes.

Pomodoro Timer

Focus timer with work/break phases.

Weather

Current weather via OpenWeather API.

RSS Feeds

Add feed URLs, auto-refresh articles.

Preset Layouts

Focus

Minimal. Large clock, search bar, quick links and tasks. For a clean new tab.

Dashboard

Full bento grid. All active widgets visible at a glance.

Workflow

Split view. Tasks on the left, stacked widgets on the right. For deep work.

Dashboard

Glassmorphism cards in a bento grid. Toggle widgets on or off per layout.

Tasks

Kanban board and list views. Add tasks from context menu or content scripts.

Bookmarks

Browse Chrome bookmarks with tree navigation. Quick search.

Settings

Customize accent color, background image, and widget toggles.
