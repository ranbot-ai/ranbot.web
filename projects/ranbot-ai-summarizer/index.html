RanBOT Labs AI Summarizer

A Chrome extension that uses OpenAI and GitLab API to summarize a GitLab issue from the issue's URL. Paste a GitLab issue URL and get an AI-powered summary.

Installation Guide

Overview

OpenAI Integration

Uses OpenAI API to generate concise summaries of GitLab issues.

GitLab API

Fetches issue content directly from GitLab.

URL-Based

Paste a GitLab issue URL to summarize.

How it works

Enter a GitLab issue URL. The extension fetches the issue and sends it to OpenAI for summarization.

Settings

Configure your OpenAI API key and GitLab settings in the extension options.

Installation

  1. Clone the repository
  2. Open chrome://extensions/
  3. Enable Developer mode
  4. Click Load unpacked and select the project folder
  5. Configure OpenAI API key and GitLab settings
