Install from Source

  1. Clone the repository
  2. Open chrome://extensions/ in Chrome
  3. Enable Developer mode (top-right toggle)
  4. Click Load unpacked and select the project folder
  5. Configure your OpenAI API key and GitLab settings in the extension options

Requirements

OpenAI API key and GitLab access token for API calls.
